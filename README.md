# Solana Anchor

A simple anchor base program on the Solana blockchain.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [pnpm](https://pnpm.io/)
- [Anchor](https://www.anchor-lang.com/)

### Installation

1. Install NPM packages

   ```sh
   pnpm install
   ```

## Usage

### Build

```sh
anchor build
```

### Deploy

```sh
anchor deploy
```

## Running Tests

To run the tests, run the following command:

```sh
anchor test
```
